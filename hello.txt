1.建立文件(git status 查看状态)
2.git add file_name
3.提交本地库：
	1.将暂存区的文件提交到本地库
	git commit -m "日志信息" 文件名
4.git reflog
5.git log
6.修改文件后，先git add,再git commit -m
7.版本穿梭：
	1.查看版本号 git reflog
	2.git reset --hard 版本号
8.分支操作：
	1.git branch 分支名	创建分支    如果使用 git branch -b 分支名 则相当于创建了分支并且切换到该分支，即1和3命令的合并
	2.git branch -v    	查看分支
	3.git checkout 分支名   切换分支
	4.git merge 分支名      把指定的分支合并到当前分支
9.远程仓库操作(github):
	1.git remote -v		查看当前所有远程地址别名
	2.git remote add 别名 远程地址		起别名	git remote rm 库名 可以删除库
	3.git push 别名 分支	 推送本地分支上的内容到远程仓库
	4.git clone 远程地址	 将远程仓库中的内容克隆到本地
	5.git pull 别名 远程分支名	远程仓库最新内容与本地仓库分支合并
